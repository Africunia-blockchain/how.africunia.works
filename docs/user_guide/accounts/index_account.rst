********************
AFRICUNIA Accounts
********************

In AFRICUNIA, you can create an account relatively easily by using the AFRICUNIA UI wallet.  The account comes with the Private/public keys. You should keep the information safe. Also, AFRICUNIA account gives you some benefits (i.e., Lifetime Membership (LTM) and Referral Program) and important roles (i.e., voting).  We recommend you to read through AFRICUNIA Accounts information to learn more.
